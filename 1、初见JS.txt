    
把与C差别较大的和自己以前没记住的写个备忘。
-------------------------------------------------------------------------------------------------
1、<script>写在<head>标签中

2、可以有多个<script>，从上到下顺序执行

3、可以引用外部文件，在script标签中加入src=“外部文件路径”。应用外部文件的script内部的代码不会执行。

4、在输出内容中用&lt代替<，%gt代替>。

5、用关键字var声明变量，不同类型的变量可以只用一个var在一行中一起声明。

6、js中变量被赋值成什么类型就是什么类型，string类型被赋值数字就会变成数字类型，可以用（typeof 变量名）查看该变量的数据类型。

7、与string类型相加，其他类型的数据会自动转化为string类型，变成字符串的拼接。js中字符串可以是汉字。

8、与string做其他运算时，string先转为数字，如果string本身不是数字串就会变成NaN，任何类型变量与NaN做运算所有结果都是NaN。

9、js代码调试，火狐/谷歌F12看console提示。

10、parseInt（）转整。操作123.45得到123；操作123a45得到123，遇到字母就结束了。
parseFloat（）转浮点。

11、js中分母可为零，结果为infinity。

12、数字和其他类型变量比较时，先统一自动转为数字

13、NaN ！= 任何值， NaN ！= NaN

14、全等 ===，不全等 ！==；
在不进行自动数据类型转换的情况下判断是否相等。

15、强制类型转换
Number（） String（） Boolean（）

16、！"" ，""表示空串，对空串进行！操作得到true

17、0、NaN、undefined的布尔值为false；
其他任何非0数字都是true，infinity也是true。

18、switch（）中可以使用任何数据类型，case的值还可以是变量。

19、换行符"<br/>"

20、定义函数要用关键字function

21、带参函数的参数声明，不需要声明类型，只写形参名字即可。

22、返回值也不需要声明类型，直接return就能返回。

23、可以不声明形参，使用时用arguments[]可以读取传入的参数。

24、也分全局变量和局部变量，作用域和C语言相同。重名时就近原则。

25、事件驱动函数（与页面交互过程中调用的函数）。通常写在window.onload = function(){内容}中。

26、按长度创建数组：var s = new Array(5)
直接赋值创建：var s = [1,"cat"]
或var s = new Array（1，"cat")
数组元素的数据类型可以不同，new可以省略

27、数组可以通过直接对末尾下一个元素赋值进行增长。s[s.length] = 新元素。

28、声明空数组 var s=[]；

29、for in 遍历
for(var item in 数组名）{}

30、栈方法 push（）返回添加后的数组长度
pop（）返回出栈元素

31、队列方法 push（）返回长度
shift（）返回出队元素
unshift(参数可以是多个)从数组头部插入元素，返回长度

32、数组合并
concat（组1，组2...）返回一个新建的合并结果数组。源数组不会被改变

33、数组部分提取
slice（起点，终点）返回一个新建的截取结果数组。
含起点位，不含终点

34、删除、插入、替换
splice（起点，长度，可选参数）
可选参数不写，就是删除；写就是在起点之前插入，可插多个，此时需要把长度置0；有长度也有可选参数就可以实现替换。

35、变字符串方法，join（拼接符）把元素拼接为字符串，拼接符选填，不填就是默认逗号拼接，可用空串""拼接。

36、变序方法：
reverse（）倒转数组；
sort（）默认按字符串大小升序；

37、字符串创建 var str = new string（“ch”）new可以省略；
var str = “hello”

38、访问某个字符 charAt（下标），也可以直接用下标访问str[i]

39、string类型不可更改，只可被覆盖。与c不同

40、获取字符的ACK码，charCodeAt（下标）

41、抽取ACK码值对应字符并组成字符串。
fromCharCode（码值1，码值2，···）

42、查找子串第一次出现的位置。
indexOf（字串，起始位置），未找到返回-1

43、找字串最后一次出现的位置
lastIndexOf（子串），未找到返回-1

44、search（）查找第一次出现的位置，参数可以是正则表达式。

45、replace（字串或正则表达式，新子串）
子串只能换第一个，正则表达式可以换全部

46、字符串提取 substring（起点，终点）
含头，不含尾。

47、字符串按分割符分割 
split（分割符，生成串组的长度）
参数2可省；分割符为""时，会分为字符数组；

48、严格模式 "use strict"
写在哪个作用域内，作用域内的变量就必须先var声明，才能使用。

49、数学计算，math对象

50、对象，js中式一种数据类型。
用new运算符创建对象：
var person = new Object（）；new可省略

51、对象的数据叫属性，函数叫方法。
person.name = "xxx"就给对象添加了一个属性
person.show = function(){}就给对象添加了一个方法

52、创建对象的另一种方法 var person = {}；
添加属性和方法的步骤是一样的
delete 可以删除属性和方法

53、日期对象
var date = new Date（）；
无参数：获取当前系统时间；
方法：用于选定输出格式
UTC世界时间。

54、周几0-6，0为周日；
月份0-11，0是一月；

55、涉及毫秒的，初始时间都是1970年

56、定时器 setInterval(函数，毫秒数);
每隔几毫秒，调用一次函数
返回值为该定时器的ID
ID用于取消定时器
var tID = setInterval();
clearInterval(tID);

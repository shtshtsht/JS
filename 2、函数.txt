函数（也是对象）

1、创建函数
var fun = new Function（）；
typeof fun；返回是function类型。

2、使用构造函数创建函数对象。（这种方法少见）
可以将需要调用执行的代码以字符串的形式传给构造函数。
var fun = new Function（“console.log('hello world')”）
把所有要执行的代码都写进去

3、函数对象具备所有对象的功能
可以设置属性值，fun.name = "新函数"；这个属性就可以和普通对象变量一样进行调用。

4、通常用函数声明的方法创建函数
function 函数名 （形参列表）{函数体}

5、使用函数表达式创建函数
var 函数名 = function（形参列表）{函数体}；（最好在末尾加上分号）
本质是创建一个匿名函数，然后把匿名函数赋值给一个变量。

6、函数调用时，不检查实参类型
即形参不用声明变量类型，直接写形参名字即可
function sum（a,b）{ }
因为语法不检查，所以在调用函数时，程序员要检查，以免出错。

7、函数调用时，不检查实参数量
多余的实参会被舍弃；
实参过少，则会有形参被赋值为undefined

8、不用考虑返回值类型，直接return即可
没有返回值，则返回undefined

9、传递的参数可以是对象
当传参数量过多，可以考虑封装到一个对象中

10、传递的参数可以是函数
函数名作实参传入

11、continue  跳过当次循环
break  跳出当前循环
return  函数结束

12、函数传入的参数可以是（任意）数据类型
返回值也可以是（任意）数据类型

13、可返回一个函数
常用于外部调用函数体内部创建的函数
function fun1(){
	function fun2(){
	}
	return fun2;
}
调用时候，fun1()();

14、匿名函数使用
匿名函数，没有对象名的函数
用小括号把匿名函数标注为整体（function（）{}）

（function（）{}）（）；此时这个函数是立即执行函数。定义完就调用。
往往只执行一次

还可以将一个匿名函数作为实参传入其他函数


15、方法
函数作为对象的属性，称这个函数为该对象的方法
